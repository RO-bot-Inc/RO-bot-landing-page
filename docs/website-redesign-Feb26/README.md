# ro-bot.io Website

Marketing website for RO-bot, the voice-first AI platform for automotive service departments.

## Status

**Planning complete.** Infrastructure files created February 8, 2026. Ready for implementation.

## Setup (When Ready to Build)

```bash
# Create new Astro project
npm create astro@latest ro-bot-website

# Add Tailwind
npx astro add tailwind

# Start dev server
npm run dev
```

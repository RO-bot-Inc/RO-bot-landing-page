# Testing Blog Images

This is a sample blog post created to test the image functionality in blog cards. The image above should appear in the blog card on the index page.

## How It Works

The system automatically extracts the first image from each blog post's markdown content and displays it at the top of the blog card, above the category badge and headline.

## Features

- Automatic image extraction from markdown
- Responsive design
- Hover effects
- Fallback for posts without images
